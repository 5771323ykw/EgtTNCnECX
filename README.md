# 社团管理系统 java273

## 项目概述

社团管理系统是一个基于B/S架构的Web网站，采用MVC模式设计。系统实现了前后端分离，前端使用Vue.js框架，后端采用Java和SpringBoot技术栈。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows
- **开发环境**:
  - IDEA
  - JDK
  - Maven
  - Mysql
- **技术栈**:
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue

## 功能模块

- 用户登录注册
- 用户修改密码
- 系统用户信息修改
- 社团加入申请
- 社团活动信息创建
- 社团活动费用记录

## 系统角色

系统主要包括以下角色：
- 普通用户：可以进行登录、注册、修改个人信息等操作。
- 社团管理员：负责审核成员申请，发布活动信息，管理活动费用。

## 运行环境

- **操作系统**: Windows
- **数据库**: Mysql
- **Web服务器**: （根据部署环境而定）

## 部署步骤

1. 准备IDEA、JDK、Maven、Mysql等开发环境。
2. 使用Maven构建项目，确保依赖项完整。
3. 部署后端服务到Web服务器。
4. 部署前端项目到Web服务器同一域名的相应目录下。
5. 配置数据库，执行数据库初始化脚本。
